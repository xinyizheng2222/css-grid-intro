Learning CSS Grid

CSS Grid is a layout system that lets you arrange content into rows and columns. In this exercise, you will practice creating grids, changing column sizes, adding gaps, and making grid items span across multiple columns.

What We Are Practicing

Each demo area below focuses on one important CSS Grid concept. The boxes are intentionally simple so you can focus on how the layout works instead of getting distracted by detailed design.

Remember: the parent element becomes the grid container when you add the display: grid; property. The direct children of that container become grid items.
